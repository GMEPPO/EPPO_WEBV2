# Documentación y scripts (no necesarios para ejecutar la web)

Esta carpeta contiene todo lo que **no es necesario** para el funcionamiento de la aplicación en el navegador:

## Scripts SQL (Supabase)
- **supabase-create-*.sql** – Creación de tablas y estructura en Supabase.
- **supabase-add-*.sql** – Migraciones para añadir columnas, triggers, etc.
- Ejecutar en el **SQL Editor** de tu proyecto Supabase cuando configures o actualices la base de datos.

## Documentación (.md)
- **README.md** – Descripción del proyecto.
- **INSTRUCCIONES-*.md** – Configuración, servidor local, bucket de fotos, etc.
- **MANUAL-UTILIZADOR-EPPO.md** – Manual de usuario.
- **ROLES-GUIDE.md**, **SEGURIDAD.md**, **VERCEL-CONFIG.md**, etc. – Notas y guías.

La web funciona solo con los archivos HTML, CSS, JS y la configuración de Supabase (URL y clave) en el código; esta carpeta es de referencia y para ejecutar scripts cuando toque.
